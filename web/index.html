Up and running in 60 seconds

  1. 01

    Install the skill

    curl -fsSL https://raw.githubusercontent.com/jfelten/ez-appsec/main/skills/install.sh \
  | bash -s -- --all

    Adds /ez-appsec-scan to Claude Code globally, plus Copilot and Cursor rules.

  2. 02

    Run a scan

    /ez-appsec-scan

    Your AI assistant pulls the Docker image (once) and runs all four scanners against your project.

  3. 03

    Fix findings with AI

    Results are grouped by severity. Ask your assistant to fix any finding and it will apply the change inline.
